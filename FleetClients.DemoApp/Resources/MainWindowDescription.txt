# Fleet Clients Demo App

Demonstrates the functionality available in the FleetClients and FleetClients.Controls assemblies.

## Configuration

Set the IP address of the scheduler to connect to in the configuration section.

## Controls Library

### Fleet Manager Client Control

A control that connects to the fleet manager and displays all the active vehicles in their fleet and associated state information.

### Fleet Template Control

A control that manipulates a fleet template which can be loaded and exported, allowing virtual vehicle fleets to be quickly instantiated.
